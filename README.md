 # Enfoque usado para cada ejercicio:

 
 # 1. Maths and Algorithms
Ejercicios enfocados en lógica matemática y técnicas de programación fundamentales.

# Enfoque utilizado

Implementación de operaciones matemáticas mediante funciones simples.
Uso de ciclos (for, while) para cálculos repetitivos.
Validación de entradas numéricas.
Algoritmos clásicos: factorial, primos, secuencias, ordenamientos básicos.
Manejo de estructuras lógicas: condicionales y bucles.

# Ejemplos de ejercicios y enfoques

Factorial: uso de acumuladores en ciclo for.
Número primo: dividir solo hasta √n para optimizar.
Fibonacci: implementación iterativa para evitar recursión costosa.
Promedios y sumatorias: manejo de contadores y acumuladores.
Ordenamientos simples: selección e intercambio básico.
Tabla de multiplicar: ciclos anidados.


# 2. Applications and Systems

Ejercicios orientados a resolver problemas cotidianos y cálculos aplicados.

# Enfoque utilizado

Conversión entre unidades (longitud, temperatura, tiempo).
Fórmulas físicas y financieras simples.
Entrada validada para datos sensibles (divisiones, porcentajes, etc.).
Separación entre lógica de cálculo y presentación.
Uso de funciones puras que reciben parámetros y devuelven resultados.

# Ejemplos de ejercicios y enfoques

Conversión de grados: fórmula estándar (°C ⇄ °F).
Cálculo de salario: manejo de porcentajes y deducciones.
Conversión de horas: uso de división entera y módulo.
Costo de servicios: lectura de datos y multiplicaciones controladas.
Velocidad, distancia y tiempo: aplicación de fórmulas físicas.


# 3. Graphics and Patterns

Ejercicios orientados a generar figuras visuales con caracteres ASCII.

# Enfoque utilizado

Uso de ciclos anidados for (filas y columnas).
Impresión usando espaciado proporcional.
Control de alineación mediante espacios ' '.
Construcción de patrones incrementales/decrementales.
Lógica de simetría (figuras centradas).

# Ejemplos de ejercicios y enfoques

Triángulos: manejo de espacios antes de los asteriscos.
Pirámides: uso de fórmulas 2n-1 para calcular ancho.
Cuadrados y rectángulos: doble ciclo estándar.
Patrones alternados: uso de índice par/impar.
Figuras huecas: validación de bordes 


# 4. Simulations and Games

Ejercicios centrados en lógica de juegos, aleatoriedad y simulación de procesos.

#Enfoque utilizado

Generación aleatoria con rand() y srand(time(NULL)).
Simulación por pasos, con estados internos.
Condiciones múltiples para definir resultados.
Lógica de turnos entre jugador y sistema.
Representación textual del progreso.

# Ejemplos de ejercicios y enfoques

Adivinar el número: bucle hasta acertar, comparaciones simple.
Dados y probabilidades: suma aleatoria y conteo.
Carrera simple: incrementos aleatorios por turno.
Batalla por turnos: variables de vida y ataque.
Simulaciones repetitivas: repetición controlada por ciclos.
